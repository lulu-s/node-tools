# node-tools
node 实用工具库存 🔧


## List
> 首先开启终端，执行 ` cd src `

### read.js
* 执行： `node read.js`
* 功能： 读取 yaml + excel 文件

### save.js
* 执行： `node save.js`
* 功能： 将数据保存为 yaml + json 格式文件

### excelToJson.js
* 执行： `node excelToJson.js`
* 功能： 将 excel 转换为 json 格式文件并保存

### jsonToExcel.js.js
* 执行： `node jsonToExcel.js.js`
* 功能： 将 json / yaml 转换为 excel 格式文件并保存

### demo.js
* 执行： `node demo.js`
* 功能：
    * 获取当前文件夹下的所有文件
